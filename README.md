# Skillway Fullstack Project

This project is organized into separate frontend and backend directories.

## Project Structure

- `backend/`: Node.js/Express server with PostgreSQL/MongoDB integration.
- `frontend/`: Static frontend files (HTML/CSS/JS).

## Getting Started

### Backend

1. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your `.env` file (see `.env` inside the `backend` folder).
4. Start the development server:
   ```bash
   npm run dev
   ```

### Frontend

The frontend consists of static files located in `frontend/public`. You can serve these using any static web server or open the `index.html` file directly in a browser.

## Technologies Used

- **Backend**: Node.js, Express, TypeScript, PostgreSQL, Mongoose
- **Frontend**: HTML5, Vanilla CSS, JavaScript
